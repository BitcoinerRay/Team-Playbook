# Launch Plan Template

## Purpose

Standard structure for launch plans: channels, copy, metrics, rollback.

## Usage

Copy `template.md` to your project (e.g. `docs/launch/<feature>.md`). Replace placeholders and fill sections.

## Structure

1. Channels
2. Copy (Headline, Key Points, CTA)
3. Metrics
4. Rollback

## Placeholders

- `{{FEATURE_NAME}}`: Feature or product name
- `{{LAUNCH_DATE}}`: YYYY-MM-DD
- `{{OWNER}}`: Launch owner

## Example

```markdown
# CSV Export - Launch Plan

**Launch Date**: 2025-03-15
**Owner**: Jane

## Channels
| Channel | Content | Timing |
|---------|---------|--------|
| In-app  | Banner  | Launch day |
| Email   | Feature announcement | Launch +1 |

## Copy
### Headline
Export your data in one click.

### Key Points
- Choose date range
- Download as CSV
- Share with team
...
```
